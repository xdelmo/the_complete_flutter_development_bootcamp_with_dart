# The Complete Flutter Development Bootcamp with Dart

Welcome to the repository for **The Complete Flutter Development Bootcamp with Dart**. This codebase is based on the [Udemy course created by the App Brewery](https://www.udemy.com/course/flutter-bootcamp-with-dart/?couponCode=MT260202G1B). This course covers all the fundamental concepts for building beautiful, high-performance mobile applications for iOS and Android using Flutter.

## ðŸ›  Getting Started

To run any of these projects locally:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/yourusername/flutter-bootcamp.git
    ```
2.  **Navigate to the project directory**:
    ```bash
    cd todoey
    ```
3.  **Get dependencies**:
    ```bash
    flutter pub get
    ```
4.  **Run the app**:
    ```bash
    flutter run
    ```
